< portfolio >

DIVYA TILES
This is a website created for one of our clients for their product. The website has a lot of jquery plugins in it to enable animations like image slider. Videos found on the site can be shared on facebook and twitter. There is a image gallery. And in regards to the back-end features the website is connected to a MySql database. When a query is typed in the search box this database is searched for a match.Besides these features, a PHP enquiry form is present to send emails directly from the website.
You can visit the website at www.divyatiles.com


NETWORK SNIFFER
I built a network sniffer for a school project. This was completely using PHP and MySql. Some major features included were socket functions to set up network between nodes of applications, GD library functions to chop images and send them through the network, file manipulation functions to determine the type of network packets.
MySql was used to store the information from the data packets, both header information and the data that were carried.


VINAYAKA MATRIMONY
This is a matrimonial website for people to find bride and bridegroom that they think will suit them. This website involves a large database of registered users. And according to the search criteria and keywords typed in results are generated to bring up the most suited profiles for the user. Besides this there are other common front-end features like slider, image gallery. The website can be found at www.vinayakamatrimony.com


CUSTOMER RELATIONSHIP MANAGEMENT TOOL
The CRM tool is a great way for the administrator to keep a watch on the users and get to know their activities better. This CRM tool was designed for an e-commerece website. This tool allowed the administrator to view user complains, add products, remove products, delete users, check the most frequented product page etc. A framework called CodeIgniter was used.


GOKARNA HOMESTAY
This is a simple website built for a resort. This website had all the basic features like jquery slider, image rotator, image gallery. For the slider a lightbox jquery plugin is used. It also has a PHP form to send emails in it's contact page. The website can be found at www.gokarnahomestay.com
